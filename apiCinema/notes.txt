O que são API's ?
 api e nada mais é do que uma forma de comunicação entre sistemas	
 as apis perm,item intregações entre dois sistemas, em que um deles fornece
informações e serviços que podem ser utilizados pelo outro, sem a necessidade
de o sistema que consome API conhecer detalhes de implementação de software

status de code
 -102 processing: processando informação
 -200 ok: listagem pronta
 -201: requisão foi bem recebida
 -204: 
 -400: nao entendeu a requisão
 -401:
 -404:
 -500: virus
